# Render CV to PDF (Local)

This skill renders any CV/resume HTML file in this repo to PDF using Puppeteer (headless Chrome), matching the same settings as the GitHub Actions workflow.

## Prerequisites

- Node.js installed (checked via `which node`)
- Google Chrome installed at `/Applications/Google Chrome.app/Contents/MacOS/Google Chrome`
- Puppeteer npm package (will be installed to temp dir if not available)

## Workflow

1. **Install Puppeteer** (if not already available):
   ```bash
   npm install puppeteer --no-save
   ```
   Run this in `/var/folders/vn/zwth_w8j2g709b8qcbh_pfyh0000gn/T/opencode` (temp dir).

2. **Create render script** at `/var/folders/vn/zwth_w8j2g709b8qcbh_pfyh0000gn/T/opencode/render_pdf.js` with content:

   ```javascript
   const puppeteer = require('puppeteer');
   const path = require('path');

   const args = process.argv.slice(2);
   if (args.length < 2) {
     console.error('Usage: node render_pdf.js <input.html> <output.pdf>');
     process.exit(1);
   }

   const inputFile = path.resolve(args[0]);
   const outputFile = path.resolve(args[1]);

   (async () => {
     const browser = await puppeteer.launch({
       headless: 'new',
       executablePath: '/Applications/Google Chrome.app/Contents/MacOS/Google Chrome',
     });
     const page = await browser.newPage();

     await page.goto(`file://${inputFile}`, { waitUntil: 'networkidle0' });

     await page.pdf({
       path: outputFile,
       format: 'A4',
       margin: { top: '7mm', left: '7mm', right: '7mm', bottom: '7mm' },
       scale: 0.95,
       printBackground: true,
     });

     await browser.close();
     console.log(`PDF generated: ${outputFile}`);
   })();
   ```

3. **Run the render script**:
   ```bash
   node /var/folders/vn/zwth_w8j2g709b8qcbh_pfyh0000gn/T/opencode/render_pdf.js <input_html_path> <output_pdf_path>
   ```

   Example:
   ```bash
   node /var/folders/vn/zwth_w8j2g709b8qcbh_pfyh0000gn/T/opencode/render_pdf.js /Users/yuehpo/coding/y10ab1.github.io/CV_sea_group.html /Users/yuehpo/coding/y10ab1.github.io/CV_sea_group.pdf
   ```

## PDF Settings (matching GitHub Actions workflow)

- Format: A4
- Margins: 7mm (top, left, right, bottom)
- Scale: 0.95
- Print background: true

## Notes

- The HTML file must use relative paths for assets (CSS, images). Since we use `file://` protocol, relative paths resolve correctly from the HTML file's directory.
- Customized CV files (e.g., `CV_sea_group.html`, `CV_sea_group.pdf`) are listed in `.gitignore` and should NOT be committed to the repo.
- Only the generic `CV.html` should be committed; its PDF is auto-generated by GitHub Actions.
